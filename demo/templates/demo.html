Library Features Demonstrated

  • feedIteration() — push iteration data
  • onComplete() — signal training done

  • stepForward() / stepBack()
  • goFirst() / goLast()

  • wireControls(elements)
  • reset() — clear & rebuild
  • DiagramConfig.to_js_config()
  • render_state_diagram() — Graphviz static
  • create_blueprint() — Flask integration
  • Inspector panel (click a state node)
  • Fullscreen mode
  • HMMDiagram backward-compat alias
